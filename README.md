# AI/ML Laboratories

This repository contains 5 laboratory assignments covering the main concepts in machine learning
and deep learning, implemented in Python using scikit-learn and TensorFlow.

## Contents

| Lab | Title | Dataset |
|-----|-------|---------|
| Lab 1 | Linear Regression | World Happiness Report |
| Lab 2 | Supervised Classification | Stars Dataset |
| Lab 3 | Clustering Algorithms | Evolution of Humans |
| Lab 4 | Artificial Neural Networks | Heart Disease (BRFSS2015) |
| Lab 5 | Convolutional Neural Networks | Marine Animals (images) |

## Brief Description

- **Lab 1** – Linear regression to predict GDP per capita from happiness indicators; 
- **Lab 2** – Comparison of Naive Bayes, SVM, Decision Tree, Random Forest, and AdaBoost; 
- **Lab 3** – K-Means, Hierarchical Clustering, DBSCAN, and EM on anthropological data; 
- **Lab 4** – Binary classification (heart disease) using Dense networks + BatchNorm + Dropout; 
- **Lab 5** – Custom CNN vs ResNet50 vs InceptionV3 for marine image classification;

## Technologies

Python · scikit-learn · TensorFlow/Keras · pandas · seaborn · matplotlib
